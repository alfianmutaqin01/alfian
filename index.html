About me

Halo, saya Alfian Mutakim, mahasiswa semester 4 di IT Telkom Purwokerto jurusan S1 Software Engineering. Saya memiliki keahlian dalam desain UI/UX, pemrograman, administrasi publik, dan copywriting. Saya mampu menggunakan tools seperti Canva dan Figma untuk desain, serta beberapa bahasa pemrograman untuk pengembangan perangkat lunak. Selain itu, saya berpengalaman dalam mengelola data dan dokumen administrasi, serta menciptakan konten yang informatif dan menarik.

Selama bekerja sebagai Admin Keuangan di Harmoni Group dan Admin serta Pelayanan Masyarakat di Kantor Kelurahan Kalikajar, saya mengembangkan keterampilan dalam pemantauan kinerja keuangan, pengarsipan dokumen, dan pelayanan masyarakat. Saya juga aktif di kampus sebagai Mentor Kewirausahaan dan Ketua Panitia Precomputition IBCF. Prestasi saya termasuk menjadi Awardee OSC Medcom.id 2021 dan Innovilage 2023 serta P2MW 2023.

Saya bersemangat untuk terus belajar dan mencari kesempatan magang di bidang administrasi, desain UI/UX, copywriting, dan IT support. Saya percaya pengalaman dan keterampilan saya akan memberikan kontribusi positif bagi perusahaan.

Services

Software Design

Saya memiliki keahlian dalam mendesain antarmuka pengguna (UI/UX) yang menarik dan intuitif menggunakan tools seperti Canva dan Figma.

Web Development

Dengan latar belakang pendidikan di bidang Software Engineering, saya memiliki kemampuan pemrograman yang cukup kuat untuk mengembangkan situs web yang fungsional dan efisien.

Graphic Design

Selain desain web, saya juga memiliki kemampuan dalam desain grafis, menciptakan berbagai materi visual yang menarik dan sesuai kebutuhan klien menggunakan Canva dan Figma.

Administrasi Keuangan

Berpengalaman sebagai Admin Keuangan di Harmoni Group Wonosobo, saya berhasil dalam pemantauan kinerja keuangan, pembuatan rekapan dan catatan data transaksi, serta pengarsipan dokumen keuangan. Saya juga terampil dalam mengelola administrasi publik di Kantor Kelurahan Kalikajar.

Copywriting

Melalui pengalaman sebagai contributing writing, saya mampu membuat konten yang informatif, menarik, dan persuasif. Pengalaman saya mencakup pembuatan konten edukatif dan realistis, serta melakukan riset dan analisis untuk menghasilkan konten yang relevan bagi audiens.

Public Speaking

Saya memiliki pengalaman dalam berbicara di depan umum, baik sebagai moderator maupun sebagai MC. Keterampilan ini saya kembangkan melalui berbagai kegiatan organisasi dan acara kampus, memastikan saya dapat menyampaikan informasi dengan jelas dan efektif.
